Entendendo a Comunicação Client x Server

Apresenta a história dos computadores e da internet. Descreve o que é um cliente, navegador, aplicações e dispositivos no vocabulário da web. Descreve o que são servidores e os seus tipos, diferenciando o que são softwares e hardwares nos servidores e a hospedagem de sites. Apresenta algumas linguagens de programação (C#, Java, Javascript e Kotlin) e mostra quais rodam no lado do servidor e quais rodam no lado do cliente, ressalta que HTML é uma linguagem de marcação e não de programação.

Introdução ao HTML

Na primeira parte mostra alguns softwares utilizados para a criação de páginas web e utiliza o inspetor de elementos do browser para mostrar a estrutura de uma página web. Inicia no VSCode – software utilizado durante o curso para a criação de páginas web - a estrutura básica de uma pagina, indicando o atalho html:5 para a criação rápida da estrutura básica de uma página html.
